STREAMFLIX

Project Overview

STREAMFLIX is a movie streaming web app designed to give users a seamless experience in watching movies and shows online.

The app offers multiple movie sources, multi-language support, and an interactive UI. Built using modern web technologies.

Technologies used:

  • HTML, CSS, JavaScript for the front-end
  • TMDb API for fetching movie data

Key Features

  • Typing Sound: A typing sound plays as the user types in the search bar.
  • Watch Button: Each movie has a "Watch" button that starts streaming the movie.
  • Multi-Language Support: Movies available with multiple dubs and captions in different languages.

Code - Typing Sound Feature

This code plays a typing sound whenever the user types in the search bar:

          
searchInput.addEventListener('input', function() {
  clickSound.play(); // Play click sound when typing in search
});

Code - Movie Search Feature

This code fetches movies from TMDb API based on the search query:

          
searchButton.addEventListener('click', function() {
  var searchText = searchInput.value.trim();
  var url = 'https://api.themoviedb.org/3/search/multi?api_key=YOUR_API_KEY&query=' + encodeURIComponent(searchText);
  fetchMovies(url);
});

Code - Watch Movie Feature

This code dynamically loads and plays a selected movie or TV series:

          
function playMovie(type, id) {
  var url = 'https://streamfr.onrender.com/api/player/' + id;
  if (type === 'tv') {
    url += '?series=1';
  }
  videoPlayer.src = url; // Set the video player's source URL
  videoContainer.style.display = 'flex'; // Show the video player
}

Conclusion

STREAMFLIX offers a user-friendly, feature-rich platform for streaming movies and shows, supporting multiple languages and dubs. The app is designed using simple technologies, making it accessible for beginners in web development.
